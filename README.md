# File Uploader

File Storage app built using Express and PostgreSQL database. The app contains user authentication and login.

## Tools

- **JavaScript** programming language
- **Node** server-side runtime
- **Expresss** backend framework
- **bcrypt** password hashing
- **PassportJS** user authentication
- **EJS** dynamic views rendering
- **PostgreSQL** database mangement
- **Prisma** database ORM
- **CSS** styling
